# @flightdeck-ai/flightdeck

Multi-agent orchestration platform for AI coding agents. Flightdeck coordinates multiple AI coding agents (Codex, Claude Code, etc.) working on the same codebase — assigning tasks, managing context, resolving conflicts, and delivering results through a unified API and dashboard.

## Quick Start

```bash
npm install -g @flightdeck-ai/flightdeck

# Initialize a new project
flightdeck init

# Start the orchestrator
flightdeck start
```

## Features

- **Project management** — Create projects, define tasks, assign to AI agents
- **Agent coordination** — Spawn, monitor, and manage multiple coding agents
- **MCP server** — Expose orchestration via Model Context Protocol
- **Decision logging** — Track all agent decisions and outputs
- **Cron scheduling** — Automate recurring tasks
- **Web dashboard & TUI** — Monitor progress in real-time
