# Retrospective

Write a retrospective after completing a spec or milestone.

## Steps
1. Use `flightdeck_task_list` to review completed tasks
2. Identify what went well and what didn't
3. Write retrospective via `flightdeck_memory_write` to `retrospectives/{date}-{topic}.md`
4. Update MEMORY.md with key lessons via `flightdeck_memory_log`

## Output Format
Save to memory:

**Retrospective: {topic}**
Completed: {date}

**What Went Well:**
- ...

**What Didn't:**
- ...

**Lessons:**
- ...

Respond with a brief summary of the retrospective.
