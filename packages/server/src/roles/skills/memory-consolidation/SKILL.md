# Memory Consolidation

Review and consolidate project memory.

## Steps
1. Read recent daily logs (last 7 days) via `flightdeck_memory_read`
2. Read current MEMORY.md via `flightdeck_memory_read`
3. Identify key insights, decisions, and patterns from daily logs
4. Update MEMORY.md with distilled learnings
5. Keep MEMORY.md under 200 lines — compress by merging related items
6. Remove outdated information

## Rules
- MEMORY.md is curated, not dumped
- Daily logs are append-only until archived
- Preserve important decisions and lessons
- Remove things that are no longer relevant

Respond with what was consolidated and any notable findings.
